sango.h

sango.h는 snagoh와 c언어의 stdio.h의 합성어이다.

스프링 부트와 AWS로 혼자 구현하는 웹 서비스

2020-04-21

Beginning

회사에서 Spring + Tomcat로 구축된 WAS에 신규 도메인을 추가할 일이 생겼다.
Java를 사용할 줄 알았기에 쉽게 할 줄 알았지만.. 현실은 코드 복붙을 하고 있는 나를 발견했다…
Spring에 대해서 공부를 해야겠다고 검색을 해보니 디자인부터 이쁜(?) 책이 있어서 사게 되었다.

summary

책 제목처럼 서비스를 구현부터 서비스를 운영하는 과정을 간략하게 체험해 볼 수 있었다.
하지만 Intellij, Spring, Gradle을 잘 모른다면 삽질을 많이 해야한다.
이동욱작가님의 Github를 보고 환경 구축을 철저하게 하자.

  • https://github.com/jojoldu/freelec-springboot2-webservice

review

2주일간 책일 보면서 실습을 진행하였지만 책의 끝까지 작성하지는 못하였다.
11개의 챕터중 9번 챕터까지만 진행을 하고 나머지 챕터들은 읽고 실습을 진행하지 않기로 했다.

물론 끝까지 할 수만 있다면 좋겠지만 나는 summary에서 언급하였듯 Intellij, Spring, Gradle, Linux, AWS 등 여기서 사용하는 다양한 기술들을 거의 처음 접했고 책에서는 간략하게 설명은 해주지만 코드의 한줄 한줄이 어떻게 돌아가는지 정확하게 이해하지 못하고 실습을 계속 진행하였다.

이 책에 대해서는 만족스럽다 비록 실습을 끝까지 진행하지는 못 하였지만 저 많은 기술들에 대해서 맛 보기도 하고 앞으로 무엇을 공부해야할지 비전(?)을 주었다.
