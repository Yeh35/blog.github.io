Hugo Blog 구축기

2020-03-11 — 1 min read
#start  #blog  #hugo 
기술 블로그를 시작한 이유 멋진 기술 블로그들을 가보면 다들 글을 왜이렇게 잘 쓰는지… 뭔가 멋있기도 하고 내가 뒤쳐저 있는 것 같기도 하고… 그냥 그런 블로그를 가지고 싶었고 시작했다. Hugo를 선택한 이유 처음에는 Tstory에 한 두개 정도 글을 썻다. 매우 편한 환경이였고 글을 맛갈나게 쓰는 것만 신경쓰면 되는 환경이였다. 하지만 한편에 아쉬움이 계속 남았다. 그래도 뭔가 더 이쁜 나만의 블로그를 가지고 싶다는 욕구가 해결되지 않았다. 친구랑 빵맥(빵에 맥주)을 하던 중 “깃 블로그 해봐"라는 말에 다음날 부랴부랴 찾게 되었다.
