2020-05-09

오늘 한 일

  • TIL(Today I learned)에 대한 블로그 글을 작성
  • 영어과외 복습 (Grammar Zone)
    • 문장 성분과 문장 형식 <– document 기록

느낀점

  • 공부를 하는 것이 생각보다 오래걸린다. (11시에 시작했는데 지금 1시다. 2시간…)
    • 물론 양에 따라 다르겠지만 한번에 정리하려고 하지말고 공부 틈틈히 정리하고 TIL은 마지막에 합치는 작업으로 해야겠다.
  • 영어 공부는 규칙 이해 -> 문장 읽기 -> 단어 암기 -> 문장 읽기 -> 단어 암기…. 반복
    • 얼마나 많은 문장을 얼마나 많은 단어를 암기하는 것이 핵심인거 같다.
    • 즉, 얼마나 오래 엉덩이를 붙이고 있느냐 + 얼마나 효율적으로 하는가에 달려있는 것 같다.
      • 효율적이라는 것은 단순작업을 줄이는 것에 있을 것 같다.
        예를 들어 모르는 단어가 나왔을 때
        1. naver 사전에 검색
        2. Excel에 단어와 뜻 복사
        3. 암기 3-1. 단어가 무슨 소리가 나는지 모르겠다면 다시 1번으로… 3-2. 예문이 궁금하다면 다시 1번으로… 여기서 Excel에 단어를 입력하기만해도 알아서 뜻과 소리를 가져온다면 단어 암기 시간을 줄일 수 있을 것이다.
      • 나중에 위에 나온걸 만들어봐야겠다..
